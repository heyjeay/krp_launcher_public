# 🛬 Kudos RP 2.0 Launcher

The official launcher for **Kudos RP** — download, install, and fly straight into the city.

## ⬇️ Download

**[Get the latest version here](https://github.com/heyjeay/krp_launcher_public/releases/latest)** — click the `Kudos-RP-2.0-Launcher-Setup-x.x.x.exe` file under **Assets**.

## 📦 Install guide

**1. If your browser complains about the download:**
- **Chrome:** click the download arrow → **Keep**
- **Edge:** `⋯` menu on the download → **Keep** → **Show more** → **Keep anyway**

**2. Run the setup.** When Windows shows the blue **"Windows protected your PC"** screen:
- Click **More info**
- Click **Run anyway**

> This appears once because we're an indie launcher, not a big-name publisher — the file is signed by us and safe. It won't ask again.

**3. Follow the installer** — it puts the **Kudos RP 2.0 Launcher** icon on your desktop.

**4. Open it and hit FLY IN** ✈️ — it cleans up any stuck GTA/FiveM, boots FiveM in pure mode, and connects you straight to the server.

## 🔄 Updates

The launcher **updates itself**. When a new version ships, a blue pill appears at the top of the launcher — one click installs it and relaunches. You never need to come back to this page.
